## Beschreibung

Das **Identity Password Hash Tool** ist ein kleines WPF-Desktop-Tool, mit dem Klartext-Passwörter für ASP.NET Identity gehasht werden können.  
Es richtet sich an Entwickler und Administratoren, die schnell und sicher Passwörter für die `AspNetUsers`-Tabelle erstellen möchten, ohne direkt auf die Webanwendung zugreifen zu müssen.

### Features
- Direkte Anzeige des generierten Passwort-Hashes  
- Saubere MVVM-Struktur, leicht erweiterbar
